S3 Storage, Redefined.

Minimal, fast, S3-compatible object storage with optional disk persistence. Built with Rust + Axum for extreme performance.

Get Started Learn More

Everything you need.

Designed for speed, reliability, and ease of use.

S3 Compatible

Works seamlessly with AWS CLI, SDKs, and any tool that speaks the S3 protocol. Support for path-style addressing.

Rust Engineered

Leverages Rust's memory safety and performance. Built on Axum for a robust, concurrent API layer.

Hybrid Persistence

In-memory speed with optional disk backup. Uses Snapshots and Write-Ahead Logs (WAL) for durability without the lag.

Multipart & Range

Handles large file uploads via Multipart and enables media streaming with fine-grained Range requests.

Built-in Observability

Prometheus metrics out of the box. Monitor your storage performance in real-time.

Security First

Optional AWS Signature V4 or Bearer Token authentication to keep your data protected.

Quick Start

Get up and running in seconds.

Run via Docker:

docker build -t vaporstore .
docker run -p 9353:9353 vaporstore

Test with AWS CLI:

export AWS_ACCESS_KEY_ID=any
export AWS_SECRET_ACCESS_KEY=any
ENDPOINT=http://localhost:9353

aws --endpoint-url $ENDPOINT s3 mb s3://my-bucket
